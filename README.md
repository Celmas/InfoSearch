# InfoSearch
## Required:
* Python 3.7.3+
* Modules:
```shell script
pip install lxml
```
```shell script
pip install requests
```
```shell script
pip install streamlit
```
```shell script
pip install pymorhy2
```
## Run crawler:
```bash
cd task1
```
```bash
python crawler
```
## Run demo:
```bash
cd task5
```
```bash
streamlit run search.py
```
